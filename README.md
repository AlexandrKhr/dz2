Задача: Создать консольную игру. Реализовать в ней возможность покупки юнитов, передвижение юнитов, возможность атаки, завершения игры.
Индивидуальное задание:  	За каждого убитого врага получаем монеты. Тратить их можно в свой ход на Ремесленников. Это сущность, дающая какое-то преимущество каким-то юнитам. Преимущество - сущность, характеризуется типом юнитов, типом улучшаемой характеристики и длительностью(в ходах). Юниты которые имеют преимущество визуально чем-то отличаются(например, с большой буквы)
